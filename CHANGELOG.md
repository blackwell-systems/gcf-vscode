# Changelog

## 0.1.0 (2026-06-14)

- Initial release
- TextMate grammar for GCF syntax highlighting
- Supports both generic and graph profiles
- Highlights: headers, section headers, summary trailers, comments, symbol IDs, edge notation, key-value pairs, scalars (numbers, booleans, null, absent), quoted strings with escapes, pipe separators, attachment markers and declarations
- File association for `.gcf` extension
